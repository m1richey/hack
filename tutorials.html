TUTORIALS

Go through these simple tutorials to give your team advantage and hit the ground running.

GitHub is used to help project teams easily share files. This is an absolute must to get familiar so that you can share code and have revision control in your back pocket before or during a hackathon! The tutorial walks you through how to install git and set up a GitHub account.

At a hackathon, you want to display your product and show it off at the end. One great way to do so can be with a website that can be hosted live and viewed by all the judges and everyone in the audience. One quick and easy way is using Google App Engine. This tutorial will help get your hackathon team up and running ahead of the pack.

Bootstrap makes front-end web development faster and easier. It's made for folks of all skill levels, devices of all shapes, and projects of all sizes. Go through this tutorial and see how we made this awesome website.
